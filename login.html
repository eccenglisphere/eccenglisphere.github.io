Conditional Sentences

Kolom diskusi pertemuan kemarin

Tujuan Pembelajaran:

Siswa dapat:

  • Mengenali dan menggunakan berbagai jenis conditional sentences (Type 0–3)
  • Menyusun kalimat pengandaian dengan struktur tata bahasa yang benar
  • Menggunakan conditional sentences dalam konteks kehidupan sehari-hari

Pengertian Conditional Sentences:

Conditional sentences (kalimat pengandaian) digunakan untuk menyatakan situasi “if” (jika/seandainya) dan akibatnya.

Tipe-Tipe Conditional Sentences:

Type Fungsi Struktur Contoh
Type 0 Fakta umum atau kebiasaan If + Present Simple, Present Simple If you heat water, it boils.
Type 1 Kemungkinan di masa depan If + Present Simple, will + base verb If it rains, we will cancel the trip.
Type 2 Situasi tidak nyata di masa sekarang If + Past Simple, would + base verb If I had more money, I would travel the world.
Type 3 Penyesalan masa lalu If + Past Perfect, would have + V3 If she had studied, she would have passed the exam.

Catatan:

  • Jangan gunakan will setelah if.
  • Gunakan were untuk semua subjek di Type 2 (contoh: If I were you…).

Worksheet: Conditional Sentences Practice

Nama: ____________________  Kelas: ___________  Tanggal: ___________

Part A: Identify the Type

Tulislah jenis conditional (Type 0–3) dari kalimat berikut:

  • If I had known about the party, I would have come. → Type: _____
  • If he works hard, he will succeed. → Type: _____
  • If you freeze water, it turns into ice. → Type: _____
  • If I were taller, I would join the basketball team. → Type: _____

Part B: Complete the Sentences

Lengkapi kalimat berikut dengan tepat:

  • If it rains tomorrow, we ___ the picnic.
  • If I see her at school tomorrow, I ___ her about the project.
  • If they don’t hurry, they ___ the beginning of the movie.
  • If she had studied harder, she ___ the test.

Part C: Make Your Own Sentences

Buatlah masing-masing satu kalimat dari tiap jenis conditional:

  • Type 1: ___________________________________________
